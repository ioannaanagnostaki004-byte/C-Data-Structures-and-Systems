# C-Data-Structures-and-Systems

This repository contains a comprehensive collection of advanced C projects. The projects range from high-level data structure implementations to low-level Unix systems programming.

---

## 🛠 Project Modules

### 1. Library Management System (Data Structures)
* **Concepts:** Linked Lists, Generic Programming (`void*`), Function Pointers.
* **Key Feature:** A modular system that evolved from static arrays to **Generic Data Structures** capable of handling any data type through callbacks.

### 2. Unix Shell & Currency Exchange Server (Networking)
* **Concepts:** Sockets (TCP), Process Forking, Pipes, File Locking.
* **Key Feature:** * A custom **Unix Shell** supporting I/O redirection and multiple pipelines.
    * A **Multi-user Server** for currency exchange using TCP Sockets and `fcntl` for database synchronization.

### 3. Parallel Auction System (IPC)
* **Concepts:** Inter-Process Communication (IPC), Shared Memory, Semaphores.
* **Key Feature:** A real-time simulation of an auction where independent processes (Bidders & Auctioneer) synchronize their state using **System V Semaphores** and **Shared Memory segments**.
